一些有用的网站或工具

发表于2024-08-18|更新于2024-08-18

数学研究相关

  1. 论文预印本平台 arXiv. 网址为 https://arxiv.org/
  2. 检索世界领域数学文献的数据库 MathSciNet. 网址为 https://mathscinet.ams.org/
  3. 通过 LaTeX 公式搜索相关数学问题的网站 Approach Zero. 网址为 https://approach0.xyz/search/

数学趣味工具与技术

  1. 尺规作图游戏 Euclidea. 网址为 https://www.euclidea.xyz/en/game/packs
  2. 数学内容可视化. 网址为 https://visualize-it.github.io/
  3. 数学动画视频制作 Manim. 网址为 https://www.manim.community/
  4. 微信公众号数学文章写作 Markdown Nice. 网址为 https://editor.mdnice.com/
  5. 介绍各种数学曲线、曲面的网站 Mathcurve. 网址为 https://mathcurve.com/
  6. 线上数学百科全书 MathWorld. 网址为 https://mathworld.wolfram.com/

LaTeX 相关

  1. LaTeX 作图网站 Mathcha. 网址为 https://www.mathcha.io/editor
  2. LaTeX 公式识别工具 SimpleTex. 网址为 https://www.simpletex.cn/ai/latex_ocr
  3. 在线 LaTeX 公式编辑器 Overleaf. 网址为 https://www.overleaf.com/
  4. TikZ 示例网站 TikZ Gallery. 网址为 https://www.integral-domain.org/
  5. LaTeX 表格生成 Tables Generator. 网址为 https://www.tablesgenerator.com/
  6. LaTeX 画交换图 quiver. 网址为 https://q.uiver.app/ (导出代码后需要添加宏包tikz-cd)
  7. LaTeX 符号识别工具 Detexify. 网址为 https://detexify.kirelabs.org/

电脑文件相关

  1. 各种文件格式转换. 网址为 https://convertio.co/zh/
  2. PDF 编辑工具 PDFgear. 网址为 https://www.pdfgear.com/online-features/
  3. PDF 编辑工具 Smallpdf. 网址为 https://smallpdf.com/cn
  4. PDF 转 SVG. 网址为 https://tools.pdf24.org/zh/pdf-to-svg
  5. 路过图床. 网址为 https://imgse.com/
  6. 寻找替代品的网站 AlternativeTo. 网址为 https://alternativeto.net/
  7. 文本文件、文档、PDF、图片等差异比较. 网址为 https://www.diffchecker.com/

人工智能相关

  1. 智谱清言 AI. 网址为 https://chatglm.cn/main/alltoolsdetail
  2. Kimi AI. 网址为 https://kimi.moonshot.cn/
  3. AI 绘图网站 Recraft (可生成矢量图). 网址为 https://app.recraft.ai/

其他

  1. 中国大学矢量校徽合集. 网址为 https://www.urongda.com/
